SonarQube Server Installation Guide

Prerequisites (All OS)

# SonarQube Community Server requirements
# - Java 17+
# - 2 vCPU+ and 4 GB RAM minimum (8 GB recommended)
# - PostgreSQL recommended for production (bundled DB only for evaluation)

Ubuntu / Debian (zip + systemd)

sudo apt update
sudo apt install -y unzip wget openjdk-17-jdk

# Create user
sudo useradd --system --create-home --shell /bin/bash sonarqube

# Download and extract
cd /opt
sudo wget https://binaries.sonarsource.com/Distribution/sonarqube/sonarqube-10.6.0.92116.zip
sudo unzip sonarqube-10.6.0.92116.zip
sudo mv sonarqube-10.6.0.92116 sonarqube
sudo chown -R sonarqube:sonarqube /opt/sonarqube

# Start manually (first run)
sudo -u sonarqube /opt/sonarqube/bin/linux-x86-64/sonar.sh start

# Access
# http://localhost:9000 (default login: admin / admin)

macOS

# Community Edition via Docker is the most stable option on macOS
brew install --cask docker

docker run -d --name sonarqube \
  -p 9000:9000 \
  sonarqube:lts-community

# Access
# http://localhost:9000

Windows

# Option 1: Docker Desktop (recommended)
winget install --id Docker.DockerDesktop -e

docker run -d --name sonarqube -p 9000:9000 sonarqube:lts-community

# Option 2: Native zip setup
# 1) Install Java 17+
# 2) Download SonarQube zip from sonarsource.com
# 3) Extract to C:\sonarqube
# 4) Run: C:\sonarqube\bin\windows-x86-64\StartSonar.bat

# Access
# http://localhost:9000
