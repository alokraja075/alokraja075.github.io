Kustomize Installation Guide

Ubuntu / Debian / RHEL / Fedora / openSUSE

curl -s https://raw.githubusercontent.com/kubernetes-sigs/kustomize/master/hack/install_kustomize.sh | bash
sudo mv kustomize /usr/local/bin/
kustomize version

Arch Linux

sudo pacman -Syu --noconfirm kustomize
kustomize version

macOS

brew install kustomize
kustomize version

Windows

winget install --id Kubernetes.kustomize -e
# or choco install kustomize -y
kustomize version

Quick Check

kubectl kustomize .
