Postman CLI Installation Guide

Ubuntu / Debian / RHEL / Fedora / Arch / openSUSE (npm method)

# Requires Node.js and npm
npm install -g @postman/cli
postman --version

macOS

brew install postman-cli
postman --version

Windows

# winget
winget install --id Postman.PostmanCLI -e

# npm alternative
npm install -g @postman/cli

postman --version

Auth & Example Run

postman login --with-api-key 
postman collection run
