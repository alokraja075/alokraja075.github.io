Argo CD CLI Installation Guide

Ubuntu / Debian / RHEL / Fedora / Arch / openSUSE (binary method)

VERSION=$(curl -s https://api.github.com/repos/argoproj/argo-cd/releases/latest | grep -Po '"tag_name": "\K.*?(?=")')
curl -sSL -o argocd "https://github.com/argoproj/argo-cd/releases/download/${VERSION}/argocd-linux-amd64"
chmod +x argocd
sudo mv argocd /usr/local/bin/
argocd version --client

macOS

brew install argocd
argocd version --client

Windows

# winget
winget install --id ArgoProj.ArgoCD -e

# choco
choco install argocd-cli -y

argocd version --client

Connect to Argo CD Server

argocd login 
argocd account get-user-info

If using self-signed cert

argocd login  --insecure --grpc-web
