kubeseal Installation Guide

Linux

curl -LO https://github.com/bitnami-labs/sealed-secrets/releases/latest/download/kubeseal-linux-amd64
test -f kubeseal-linux-amd64 && chmod +x kubeseal-linux-amd64 && sudo mv kubeseal-linux-amd64 /usr/local/bin/kubeseal
kubeseal --version

macOS

brew install kubeseal
kubeseal --version

Windows

winget install --id BitnamiLabs.kubeseal -e
# or choco install kubeseal -y
kubeseal --version
