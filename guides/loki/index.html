Grafana Loki Installation Guide

Linux (all distros, binary method)

curl -LO https://github.com/grafana/loki/releases/latest/download/loki-linux-amd64.zip
unzip loki-linux-amd64.zip
chmod +x loki-linux-amd64
sudo mv loki-linux-amd64 /usr/local/bin/loki
loki --version

macOS

brew install loki
loki --version

Windows

# Download loki-windows-amd64.zip from releases page
# Extract and run:
loki-windows-amd64.exe --version

Quick Start

loki -config.file=loki-local-config.yaml
