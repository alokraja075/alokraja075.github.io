Kind Installation Guide

Ubuntu / Debian / RHEL / Fedora / openSUSE

curl -Lo ./kind https://kind.sigs.k8s.io/dl/latest/kind-linux-amd64
chmod +x ./kind
sudo mv ./kind /usr/local/bin/kind
kind version

Arch Linux

sudo pacman -Syu --noconfirm kind
kind version

macOS

brew install kind
kind version

Windows

winget install --id Kubernetes.kind -e
# or choco install kind -y
kind version

Quick Start

kind create cluster
kubectl cluster-info
